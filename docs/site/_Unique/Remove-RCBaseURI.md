# Remove-RCBaseURI

## SYNOPSIS
Removes the RocketCyber base URI global variable.

## SYNTAX

## DESCRIPTION
The Remove-RocketCyberBaseURI cmdlet removes the RocketCyber base URI global variable.

## EXAMPLES

### EXAMPLE 1

Removes the RocketCyber base URI global variable.
