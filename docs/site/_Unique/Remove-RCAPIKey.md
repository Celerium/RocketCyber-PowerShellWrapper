# Remove-RCAPIKey

## SYNOPSIS
Removes the RocketCyber API bearer token global variable.

## SYNTAX

## DESCRIPTION
The Remove-RocketCyberAPIKey cmdlet removes the RocketCyber API
bearer token global variable.

## EXAMPLES

### EXAMPLE 1
```powershell
Remove-RocketCyberAPIKey
```

Removes the RocketCyber API bearer token global variable.
